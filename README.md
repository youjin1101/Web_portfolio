# ⚾ 갈매기 야구 마켓

롯데 자이언츠 팬들을 위한 야구용품 및 굿즈 쇼핑몰 웹사이트입니다.

HTML, CSS, JavaScript를 활용하여 제작하였으며, 로그인 및 장바구니 기능을 구현하였습니다.

---

## 📌 프로젝트 소개

갈매기 야구 마켓은 야구 팬들이 유니폼, 응원도구, 글러브, 배트 등 다양한 상품을 확인하고 장바구니에 담을 수 있는 쇼핑몰 웹사이트입니다.

반응형 웹 디자인을 적용하여 PC와 모바일 환경 모두에서 사용할 수 있도록 구현하였습니다.

---

## 🛠 사용 기술

* HTML5
* CSS3
* JavaScript
* LocalStorage

---

## ✨ 주요 기능

### 로그인 / 로그아웃

* 지정된 계정 정보로 로그인
* LocalStorage를 이용한 로그인 상태 유지
* 로그아웃 기능 구현

### 상품 페이지

* 상품 목록 조회
* 상품 장바구니 담기
* 동일 상품 추가 시 수량 증가

### 장바구니

* 로그인 사용자만 이용 가능
* 상품 수량 증가 / 감소
* 상품 삭제
* 총 금액 자동 계산
* 주문하기 기능

### My Page

* 로그인 사용자 정보 표시
* 로그인 상태에 따른 화면 변경

### 반응형 웹

* 모바일 햄버거 메뉴
* 모바일 전용 사이드바
* 화면 크기에 따른 레이아웃 변경
